## EPIM 到底在做什么

一句话概括：把你工作和生活里散落的"信息碎片"，归拢到一个本地数据库里统一管。

具体到模块，它涵盖邮件、日历、任务、笔记、联系人、密码这几大块。邮件支持 POP3 / IMAP / Microsoft Exchange 协议，可以挂多个账户统一收发；日历支持 Google、iCloud、CalDAV 同步；笔记支持 RTF 和 Markdown 两种格式；数据用 AES-256 加密存储在 Firebird 数据库里，本地优先，不上云也能跑。

它和 Outlook 的差别在于：Outlook 是 Office 生态的一部分，离开 Exchange / Microsoft 365 体验会打折；EPIM 则是一个独立的、本地优先的 PIM，不绑定任何云服务，数据文件就是后缀为 `.epim` 的一个文件，你可以随便拷贝、备份、迁移。

它和 Notion 的差别更明显：Notion 是云端协作工具，强在团队共享；EPIM 是单人 / 小团队的本地信息中心，强在数据自持和离线可用。

**支持哪些平台？** EPIM Pro 主程序是 Windows 版。iOS 和 Android 有对应的移动端 App，可以和桌面端通过 EPIM Cloud 或第三方云服务同步数据。macOS 没有原生桌面版。

**数据安全吗？** 数据默认本地存储，AES-256 加密，数据库是 Firebird。如果你不开启云同步，数据不会离开你的设备。开启 EPIM Cloud 同步的话，传输走 SSL 加密。

**30 天能退款吗？** 能。EPIM 官方对 Pro 和 Server 版本提供 30 天退款保证，不满意可以退。

**和用盗版、破解版比呢？** 盗版和破解补丁里经常夹带木马、后门或恶意脚本，存在隐私泄露和设备安全风险。异常低价的注册码也可能涉及信用卡盗刷或共享授权滥用，被开发商识别后会封禁，退款和售后都麻烦。EPIM 官方提供免费版和 30 天试用版，即便暂不买正版，也建议从官方渠道获取。
